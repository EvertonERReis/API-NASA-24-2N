# NASA APOD Viewe 🚀

Uma aplicação web construida **REACT** e **Vite** que consome a API **Astronomy
Picture of they Day (APOD)** da NASA. O projeto exibe diariamente uma imagem (ou vídeo) do universo, acompanhando de título, descrição e outras informações.

## Tecnologias Utilizadas 

- REACT
- VITE
- JAVASCRIPT (ES6+)
- API pública da NASA (APOD)

  ## funcionalidades
  - Exibição de imagens e vídeos do dia
  - Título, descrição e data
  - Indicador de carregamento
